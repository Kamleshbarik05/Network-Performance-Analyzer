# Network Performance Analyzer

This is a Python project that analyzes network performance.

It checks:
- Network latency
- Internet speed
- Bandwidth usage
- Real time graph visualization of latency

## Features
- Network latency testing
- Internet speed testing
- Bandwidth monitoring
- Latency graph visualization

## Technologies Used
- Python
- ping3
- psutil
- speedtest-cli
- matplotlib

## How to Run
Run the program using:
python main.py
